LBO Returns Calculator

Estimate sponsor returns from a leveraged buyout using purchase price, financing mix, EBITDA growth, debt paydown, and exit multiple assumptions.

How LBO returns work An LBO return depends on the entry equity check, operating growth, debt paydown, and exit valuation. The sponsor wins when exit equity value grows faster than the initial equity invested.
See the formula
MoIC = Exit Equity Value / Initial Sponsor Equity

IRR is the annualized return that links the initial equity investment to the exit equity proceeds over the holding period.

How to Use This Calculator

1. Set Entry Value

Start with EBITDA and an entry multiple to estimate purchase enterprise value.

2. Size the Debt

Use a debt multiple and cash conversion assumptions to model deleveraging during the hold.

3. Check Exit Returns

Compare MoIC and IRR across multiple scenarios to see what drives the sponsor return.
